What is Chiplotle?¶

Chiplotle is a Python library that implements and extends the HPGL (Hewlett-Packard Graphics Language) printer control language. It is written and maintained by Víctor Adán and Douglas Repetto.

Find all there is to know about Chiplotle at: http://music.columbia.edu/cmc/chiplotle
